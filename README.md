## 🚀 Quick Start

### Option 1: Open Directly
Simply open `index.html` in your web browser.

### Option 2: Local Server (Recommended)

**Using Python:**
```bash
cd Adarsh-Portfolio
python -m http.server 8000
```

**Using Node.js:**
```bash
npx serve
```

Then open `http://localhost:8000` in your browser.
